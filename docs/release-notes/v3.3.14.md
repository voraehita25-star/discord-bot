# v3.3.14 — Security Hardening & Reliability Fixes

**Date:** 2026-03-13

## Security Fixes (6)
- **AllowedMentions default** — Bot now blocks `@everyone`/`@here` mentions by default + inline sanitization in AI send paths (`bot.py`, `logic.py`)
- **Path traversal in `safe_delete()`** — Validates path is within `temp/` directory (`cogs/music/cog.py`)
- **SQL injection in `increment_user_stat()`** — Whitelist dict instead of f-string column names (`database.py`)
- **Silent guardrails failure** — Logs `critical()` instead of silently disabling safety (`logic.py`)
- **Token/API key leakage in logs** — Added `SensitiveDataFilter` with regex patterns (`logger.py`)
- **URL cache race condition** — Added `asyncio.Lock` for cache operations (`url_fetcher.py`)

## Reliability Fixes (5)
- **Circuit breaker mixed locks** — Removed `asyncio.Lock`, unified to `threading.Lock` only (`circuit_breaker.py`)
- **Music queue no locking** — Added per-guild `asyncio.Lock` (`queue.py`)
- **error_recovery double-locking** — Removed redundant `asyncio.Lock` (`error_recovery.py`)
- **Self-healer false positives** — Exact filename matching via `PureWindowsPath.name` (`self_healer.py`)
- **`asyncio.TimeoutError` not caught on Python 3.10** — `except (TimeoutError, asyncio.TimeoutError)` (`dashboard_chat.py`)

## Test Fixes (2)
- **`test_dashboard_chat.py` mock spec** — Use real `genai_types.Part`/`Content` instead of `MagicMock` 
- **`test_thought_as_string`** — Use `SimpleNamespace` for edge-case (thought as string)
