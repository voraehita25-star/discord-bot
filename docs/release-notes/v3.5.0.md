A minor release rolling up everything since v3.4.14: two complete dashboard
visual redesigns capped by **Sakura Midnight v4**, the Claude CLI backend moved

## Embedded chat isolation (privacy)

The dashboard's embedded `claude -p` chat no longer runs inside the repo:

- **Workdir moved out of the repository** — the spawned CLI can no longer read
  the project `CLAUDE.md` or git state into a persona conversation.
- **Isolated from the operator's Claude Code auto-memory** — persona sessions
  and the operator's dev sessions no longer share memory state.
- Spawned sessions are isolated per-conversation and session logs are
  sync-deleted with their conversation.

## Claude CLI backend
